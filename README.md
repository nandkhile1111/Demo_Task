# Expense Tracker Demo (Django + PostgreSQL)

## Description
A simple Django web application demonstrating:
- CRUD operations for expenses via REST API
- Integration with a third-party Weather API
- Expense reporting with charts using Chart.js

## Features
1. Expenses CRUD API
2. Weather API
3. Expense report with charts

## Setup Instructions
1. Clone repo
2. Create virtual environment
3. Install dependencies
4. Apply migrations
5. Run server

## API Endpoints
- Expenses CRUD: http://127.0.0.1:8000/api/expenses/
- Weather API: http://127.0.0.1:8000/api/weather/Delhi/
- Expense Report: http://127.0.0.1:8000/api/report/
